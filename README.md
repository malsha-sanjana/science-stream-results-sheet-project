# Science Stream Results Sheet Project

A Python project to calculate A/L Science stream examination results.

## Features
- Calculate subject grades
- Calculate subject-wise Z-scores
- Calculate final average Z-score
- Validate user mark inputs

## Supported Subjects
- Combined Mathematics
- Physics
- Chemistry
- Biology
- Information and Communication Technology (ICT)
- Agricultural Science

## Grade System
| Marks | Grade |
|------|-------|
| 75+ | A |
| 65-74 | B |
| 55-64 | C |
| 35-54 | S |
| Below 35 | F |

## Z-Score Formula
Z = (X - μ) / σ

Where:
- X = Student marks
- μ = Subject average
- σ = Standard deviation

- Python
- Functions
- If/Else
- While Loop
- Try/Except
